# gitconfig

My git configuration — delta diff viewer with catppuccin-mocha theme, useful aliases, and sensible defaults.

## What's included

- Delta as git pager with side-by-side diff, catppuccin-mocha colors
- Extensive aliases (log variants, interactive staging, etc.)
- Git LFS support
- Personal info split to `~/.gitconfig.local` (not tracked)

## Install

```bash
./install.sh
```

After install, edit `~/.gitconfig.local` with your name and email.
